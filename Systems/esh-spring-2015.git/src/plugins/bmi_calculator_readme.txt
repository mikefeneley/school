The 'bmi index calculator' esh plug-in
-----------------------

COMMANDS:

	'bmi <height in meters> <weight in kg> ' calculates the BMI index and gives result

	eg: 'bmi 1.70 70' will give
			Your BMI result is = 24.22

The 'bmi Index calculator' plug-in calculates the bmi index using the given heigh and weight information.
